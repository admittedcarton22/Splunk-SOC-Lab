# Splunk SOC Lab

## Overview

This project demonstrates my ability to use Splunk Enterprise as a SIEM to detect, investigate, and analyze Windows security events.

## Environment

- Windows 11 VM
- Splunk Enterprise
- Splunk Universal Forwarder
- Sysmon
- VirtualBox

## Skills Demonstrated

- SPL
- Windows Event Logs
- Threat Hunting
- Log Analysis
- Dashboard Creation
- Incident Investigation
